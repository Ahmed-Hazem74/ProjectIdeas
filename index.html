⚡ First-time Setup

This app uses JSONBin.io as a free shared database so all 7 team members see the same data. You need to create a free bin and paste the Bin ID below. This is a one-time setup.

Steps: Go to jsonbin.io → Sign up free → Create a new bin with this initial JSON:

{"members":[{"username":"admin","title":"Project Admin","role":"admin"}],"ideas":[]}

Then copy the Bin ID (looks like 64abc1234…) and your Master Key, and paste below.
